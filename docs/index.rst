Sugar Project
=============

Welcome to Sugar Project!

What is Suger? Sugar is a distributed and event driven systems
management tool prototype. Its main goal is to take care of next
generation infrastructure layouts that contains microservices
and immutable systems.

And remember: sugar is included in cookies!
